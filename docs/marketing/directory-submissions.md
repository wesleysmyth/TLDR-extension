## Chrome Extension Directories

### 1. Chrome-Stats

- [ ] Submitted

**URL:** https://chrome-stats.com/
**What it does:** Tracks Chrome extension statistics (installs, ratings, ranking over time). Having your extension indexed here increases discoverability and provides a backlink to your CWS listing.
**What to submit:** Your extension is auto-indexed from the CWS, but you can claim your listing for additional features. Visit https://chrome-stats.com/ and search for your extension by name or ID. If it does not appear yet, submit your CWS URL.
**Info needed:**
- CWS extension URL
- Developer email (for claiming the listing)

**Tips:**
- Chrome-Stats typically indexes new extensions within a few days of CWS publication.
- Claiming your listing lets you track competitor stats and keyword rankings.
- Check back after 48 hours if your listing does not appear immediately.

**Optimal timing:** Submit on Day 2 (after CWS listing is live with updated metadata).

### 2. CRXcavator

- [ ] Submitted

**URL:** https://crxcavator.io/
**What it does:** Runs automated security analysis on Chrome extensions and publishes a report. A good security score builds trust with power users and enterprise buyers. The report page creates a high-authority backlink.
**What to submit:** Extensions are typically auto-indexed. Search for your extension at https://crxcavator.io/ by name or extension ID (`hmphaahhfmfdebdjedigdmjjnbcdpjkm`). If not found, submit your CWS URL for analysis.
**Info needed:**
- CWS extension ID

**Tips:**
- TLDR requests minimal permissions (activeTab, storage, scripting, tabs) which should yield a strong security score.
- Link to your CRXcavator report from your GitHub README once it is available -- this signals trust to developers.
- If your score is low on any dimension, address the issue and resubmit.

**Optimal timing:** Submit on Day 2. Analysis usually completes within minutes.

### 9. Reddit

- [ ] r/chrome posted
- [ ] r/ChatGPT posted
- [ ] r/productivity posted
- [ ] r/InternetIsBeautiful posted

**URL:** https://www.reddit.com/
**What it does:** Reddit drives significant traffic when posts gain traction. Each subreddit requires a different angle to resonate with that specific community.
**What to submit:** Tailored posts for each subreddit (see launch-playbook.md for full templates).

**Subreddit breakdown:**

**r/chrome** (~300K members)
- Angle: "I built a useful Chrome extension" -- this community loves discovering new extensions
- Title format: "I built a free Chrome extension that summarizes articles in 2 seconds"
- Include: CWS link, screenshot, brief description

**r/ChatGPT** (~5M members)
- Angle: "I solved a ChatGPT workflow pain point" -- position as a time-saver for ChatGPT users
- Title format: "I built a Chrome extension so you don't have to copy-paste articles into ChatGPT"
- Include: Before/after workflow comparison, CWS link

**r/productivity** (~2M members)
- Angle: Results and time-savings -- this community cares about measurable productivity gains
- Title format: "I saved 2+ hours per week with this article summarizer I built"
- Include: Specific time-savings example, workflow description, CWS link

**r/InternetIsBeautiful** (~17M members)
- Angle: Share the landing page as a useful web resource
- Title format: Landing page URL as the link post
- Note: This subreddit only allows link posts. Submit the landing page URL directly.

**Tips:**
- Do NOT post to multiple subreddits on the same day. Spread them across Day 4 and Day 5.
- Reddit users are hostile to overt self-promotion. Frame every post as "I built this and want to share it" not "check out my product."
- Engage authentically with every comment, even critical ones.
- Post between 8 AM and 10 AM ET on weekdays for maximum visibility.
- If a post gets removed, do not repost. Message the moderators politely.
- Do NOT ask for upvotes anywhere. Ever.

**Optimal timing:** Day 4 of launch (stagger posts across Day 4-5).
