## Privacy Practices

### Single Purpose Description
This extension summarizes web articles using AI, allowing users to quickly understand article content without reading the full text.

### Permission Justifications

**activeTab**: Required to read article content from the current page for summarization.

**storage**: Required to save user preferences (API key, tone/length/focus settings) and cache summaries locally.

**Host Permission (all URLs)**: Required to extract article content from any website the user visits. The extension only activates when the user clicks the icon.

### Data Usage

**What data is collected:**
- Article content (title, text) is sent to Groq's API for summarization
- No personal information is collected

**How data is used:**
- Article content is processed by Groq to generate summaries
- Summaries are cached locally for faster loading

**What data is shared:**
- Article content is shared with Groq (AI provider) for processing
- No data is shared with any other third parties

**Data retention:**
- Summaries are cached locally until the user clears them
- No data is retained on external servers
