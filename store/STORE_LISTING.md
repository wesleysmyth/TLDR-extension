## Privacy Practices

### Single Purpose Description
TLDR - AI Article Summarizer for Web & News summarizes web articles using AI, allowing users to quickly understand article content without reading the full text.

### Permission Justifications

**activeTab**: Required to read article content from the current page when the user clicks the extension icon to request a summary.

**storage**: Required to save user preferences (API key, style settings) and cache summaries locally for faster repeat access.

**Host Permission (all URLs)**: Required to extract article content from any website. The extension only activates when explicitly clicked by the user - it never runs automatically or in the background.

### Data Usage Disclosure

**What data is collected:**
- Article title and text content (only when user clicks to summarize)
- No personal information, browsing history, or tracking data

**How data is used:**
- Article content is sent to Groq's API to generate a summary
- Summaries are cached locally in browser storage

**What data is shared:**
- Article content is shared with Groq (groq.com) for AI processing
- No data is sold or shared with advertisers
- No analytics or tracking services are used

**Data retention:**
- Summaries cached locally until user clears them
- No data retained on external servers after processing
